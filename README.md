# SneekSave Web App
SneekSave is a web application built with an aim of helping the users in downloading videos, clips and reels easy 

----
# The Mission

SneekSave is a high-performance web application built with the aim of demystifying media discovery and personal archiving. In an era where content is fragmented across thousands of platforms, SneekSave acts as a unified, stealthy portal. It allows users to search for their favorite content, preview it instantly, and save it for offline useâ€”all without the clutter, tracking, or invasive ads typical of traditional downloaders.

----
# Why SneekSave?

1. Search-First Workflow: No more tab-switching. Use the built-in search engine to find content directly across multiple platforms.
2. Stealthy Speed: Powered by a Rust-optimized backend (uv) and a cutting-edge React 19 frontend for near-instant responses.
3. Quality & Control: Choose your resolution, trim clips to the exact second, or extract high-fidelity audio (MP3) with one click.
4. Privacy Centric: No trackers, no bloat. Just the media you want, delivered to your device.

# The Tech Stack
----
1. Frontend: React 19 + Tailwind CSS 4 + Vite (Managed with pnpm)
2. Backend: Flask + REST-X + SQLAlchemy (Managed with uv)
4. Engine: yt-dlp & FFmpeg for rock-solid media extraction.
